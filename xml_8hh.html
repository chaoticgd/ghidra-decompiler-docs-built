Send the given character array to a stream, escaping characters with special XML meaning.

This makes the following character substitutions:

  • '<' => "&lt;"
  • '>' => "&gt;"
  • '&' => "&amp;"
  • '"' => "&quot;"
  • '\'' => "&apos;"
